# 共享小程序集合（童车-轮椅-陪护床等）

#### 介绍
共享小程序集合，可以运营一切IoT智能物联网锁的共享项目（如：共享童车、共享轮椅、共享陪护床、共享购物车、共享单车、共享景区车、智能售货柜）。已对接微信支付、微信信用分免押支付，已对接IoT物联网锁。

特别说明：满足以下应用场景和模式即可直接使用，只需要后台重新上传素材和数据即可（如：将轮椅换成陪护床），无需修改代码。

#### 软件架构
基于JeecgBoot开发，支持SpringBoot和SpringCloud随时切换部署，满足后期访问量大的需求！

#### 使用说明

1.  注册微信小程序并通过认证、开通微信支付、开通微信信用分免押支付；
2.  选择合适的一款IoT物联网智能锁（见下图）；
3.  开通云ECS服务器；
4.  安装宝塔面板，并配置域名、SSL证书、Nginx、MySQL、Redis等；
5.  启动后端服务；
6.  发布小程序和H5网页；
